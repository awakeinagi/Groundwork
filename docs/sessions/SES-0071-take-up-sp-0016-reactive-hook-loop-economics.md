Findings: (A) Convergence to a fixpoint -- an IDEMPOTENT behavior (mark a dependent only if it is not already stale) converges to runtime idle on ALL topologies including a CYCLE: chain(100 nodes) -> 99 marked in 100 firings; tree(127) -> 126 in 127; cycle(50) -> 50 in 51 firings; each node fires once, events burned are O(reachable nodes).
