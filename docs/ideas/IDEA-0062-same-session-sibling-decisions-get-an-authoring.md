# IDEA-0062: Same-session sibling decisions get an authoring-time cross-link check

## The Idea

An authoring-time check — when a session produces multiple decisions on the same subject surface or sharing a primary citation target, require or prompt mutual relates-to links at distillation, so the gap is closed when the decisions are recorded rather than caught retroactively by recall audits.

## Spark Context

Both recall-audit judges for DEC-0460/DEC-0461 independently surfaced the same contract gap: sibling decisions born in the same session that share a citation target (both cited only DEC-0074) escape the DEC-0157 relates-to sweep, because the sweep walks citers of already-named relations and has no edge between un-cross-linked siblings. Raised and ratified at SES-0088 close.
