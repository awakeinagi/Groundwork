# IDEA-0002: The Idea artifact type in the application

## The Idea

SES-0050 added a paradigm-level Idea artifact type (DEC-0258:
`IDEA-` / `docs/ideas/`, captured → taken-up | declined; DEC-0259:
pre-classification boundary; DEC-0261: work-queue semantics). The
application should reflect it: a `SPEC-idea.md` artifact spec, the
artifact-type enum in schemas and storage, Idea capture/browse in the
UI, work-queue surfacing beside untriaged CPs (the triage views of
ST-0055 are the natural neighbor), and take-up flow opening an intake
session from a captured Idea.

## Spark Context
