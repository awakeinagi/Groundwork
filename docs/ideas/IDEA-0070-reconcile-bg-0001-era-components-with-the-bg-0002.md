# IDEA-0070: Reconcile BG-0001-era components with the BG-0002 epic architecture

## The Idea

Reconcile the BG-0001-era epic and component set (all 16 CMP-* artifacts, including CMP-0004's monolithic governance evaluation) against the BG-0002 epic architecture and DEC-0423's single-sourcing mandate — the separate future step DEC-0441 named but nothing yet tracks. Today all 16 components derive solely from BG-0001-era epics, zero reference the new epic set, and the corpus holds two competing pictures of where governance evaluation lives (CMP-0004's monolith vs. the EP-0010/EP-0014/EP-0015 three-way split).

## Spark Context

Surfaced by the SES-0094 decision-recall audit on EP-0014 (judge contract-gap note on CMP-0004/DEC-0162). Investigated in-session: no accepted decision maps old components to the new architecture, and no CMP has been reconciled. The stakeholder dispositioned it as a parked idea for a future BG-0001-reframing session rather than in-scope for EP-0014.

## Disposition

Pending.
