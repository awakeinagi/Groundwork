# IDEA-0063: Align the product-side elaborate affordance with DEC-0460's method-level copy

## The Idea

Consider aligning the Refinement Web UI's elaborate-affordance copy and threshold with DEC-0460 so product and method stay in step: the verbatim label "Please elaborate on what needs to be decided here, include examples where helpful," a listed option only on cards with at most two substantive alternatives, a keyword fallback otherwise, and implicit-elaborate treatment of "I don't understand"-style answers.

## Spark Context

DEC-0074 and ST-0044 promise an always-present elaborate affordance in the Refinement Web UI's question cards, worded and shaped independently of the method-level practice DEC-0460 now defines for facilitator sessions. This is a product-scope refinement touching approved artifacts (ST-0044, DEC-0074), so it would run its own session with the DEC-0267 machinery if taken up. Raised and ratified at SES-0088 close.
